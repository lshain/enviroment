Emacs 修改exec-path

分类： Emacs 2012-06-16 17:04 47人阅读 评论(0) 收藏 举报 


由于当初安装texlive的时候没有使用默认目录，故命令所在目录不在/usr/bin等目录下导致Emacs Eshell中找不到xelatex命令。




添加到.emacs中：/home/visayafan/VfInstall/texlive/2011/bin/i386-linux/为你要执行命令所在的目录，可以通过which命令查看。


[html] view plaincopyprint?
  1. (setenv "PATH" (concat (getenv "PATH") ":/home/visayafan/VfInstall/texlive/2011/bin/i386-linux/"))  
  2. (setq exec-path (append exec-path '("/home/visayafan/VfInstall/texlive/2011/bin/i386-linux/")))  
(setenv "PATH" (concat (getenv "PATH") ":/home/visayafan/VfInstall/texlive/2011/bin/i386-linux/"))
(setq exec-path (append exec-path '("/home/visayafan/VfInstall/texlive/2011/bin/i386-linux/")))
